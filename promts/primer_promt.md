Por favor, lee detalladamente el archivo AGENTS.md para asimilar el contexto, la arquitectura, el stack tecnológico y las directivas de UI/UX del proyecto 'Stark Commerce OS'.

Como nuestra primera iteración, necesito que ejecutes lo siguiente:

Inicializa el proyecto en Next.js 16 (App Router) con TypeScript y Tailwind CSS.

Configura Prisma y crea el archivo schema.prisma definiendo exactamente los modelos Product, Purchase y Sale con las relaciones especificadas en el AGENTS.md.

Configura las variables de entorno para conectar con Supabase.

Crea la estructura base de carpetas y rutas (app/) para los 5 módulos principales: /productos, /compras, /ventas, /inventario y /dashboard.

Crea un layout.tsx principal que incluya una barra de navegación inferior (Bottom Navigation Bar) adaptada para dispositivos móviles Android que permita navegar entre estos 5 módulos.

No te preocupes por la lógica de base de datos todavía; enfócate en el scaffolding del proyecto, el esquema de Prisma y la estructura de rutas.
